# Responsive Landing Page

A modern responsive landing page built using **HTML5** and **CSS3**. This project was created to strengthen my understanding of responsive web design and modern CSS layout techniques.

## 🚀 Features

- Responsive layout
- CSS Grid
- Flexbox
- Hero section with overlapping elements
- Gradient backgrounds
- Course cards
- Responsive navigation
- Footer section

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Media Queries
- CSS Positioning

## 📚 What I Learned

During this project I practiced:

- Creating responsive layouts
- Using CSS Grid and Flexbox together
- Relative and absolute positioning
- Building card-based layouts
- Applying gradients and border-radius
- Structuring HTML semantically
- Writing cleaner CSS
